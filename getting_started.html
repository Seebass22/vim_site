vimtutor

If you've never used vim before, then the first thing you should do is go through the built-in tutorial. Just run vimtutor from the command line if using vim. Or if you're using neovim, you can start the tutorial by typing :Tutor from within neovim. This will teach you the basics and should take around half an hour.

the user manual

When you feel like learning more about vim, go through the user manual (:help usr_01.txt) at your own pace. Skip over anything you don't need.
