commentary

github.com/tpope/vim-commentary
For some reason, vim does not have a feature for quickly commenting-out lines. This simple plugin fixes that.

dispatch

github.com/tpope/vim-dispatch
Vim's :make feature is not asynchronous by default. That means that while make is running, you won't be able to use your editor and you'll have to wait for make to finish before you can continue. Dispatch solves this problem by running make in tmux. If you're using neovim, you can make dispatch use the neovim terminal instead of tmux by additionally using this plugin.

fugitive

github.com/tpope/vim-fugitive
Fugtive is a git-wrapper for vim. It makes using git from within vim easier, for example by automatically deleting the right buffers when you run git rm.
