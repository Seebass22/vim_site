compiling using makefiles

Vim has support for running makefiles and showing location of errors and warnings resulting from your compilers output (See :help usr_30.txt).

jumping to function definitions

Vim can jump to function definitions as long as you have a tag file. To generate this file you'll need to install ctags (not included with vim). Once you've generated your tags file e.g. by running ctags *.c, you can jump to function declarations by pressing CTRL-], while the cursor is on a function name (see :help usr_29.txt)
